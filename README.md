# AI-Based Real-Time Driver Cognitive Monitoring System

## Overview
This project is a computer vision-based driver fatigue monitoring system designed to analyze driver alertness in real time using facial landmark analysis and fatigue detection techniques.

The system aims to improve road safety by monitoring driver behavior through webcam-based cognitive state estimation.

---

## Features

- Real-time webcam monitoring
- Driver fatigue detection
- Eye blink analysis
- Facial landmark tracking
- Real-time alert system
- Scalable AI pipeline

---

## Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy

---

## Planned AI Enhancements

- Personalized fatigue prediction
- Temporal behavior analysis

- Real-time analytics dashboard
